## 1. 工具函数

### 1.1 动态对象加载器

**用途**：从资产路径动态加载任何类型的对象

**代码**：
```cpp
/**
 * 从资产路径动态加载对象
 * @tparam ObjType 要加载的对象类型
 * @param Path 资产路径（如 "Texture2D'/Game/Textures/MyTexture'"）
 * @return 加载的对象指针，失败返回nullptr
 */
template <typename ObjType>
static FORCEINLINE ObjType* ObjectFromPath(const FName& Path)
{
    if (Path == NAME_None) return nullptr;
    return Cast<ObjType>(StaticLoadObject(ObjType::StaticClass(), nullptr, *Path.ToString()));
}

// 使用示例
UBehaviorTree* AITree = ObjectFromPath<UBehaviorTree>(TEXT("BehaviorTree'/Game/AI/BT_Enemy'"));
UTexture2D* Texture = ObjectFromPath<UTexture2D>(TEXT("Texture2D'/Game/Textures/Crosshair'"));
USoundCue* Sound = ObjectFromPath<USoundCue>(TEXT("SoundCue'/Game/Sounds/Gunshot'"));
```

**复用指数**: ⭐⭐⭐⭐⭐  
**适用场景**: 所有需要动态加载资产的UE项目

### 1.2 DataTable行加载器

**用途**：从DataTable加载特定行的数据

**代码**：
```cpp
/**
 * 从DataTable按路径和行名检索行
 * @tparam DatatableType DataTable行的结构体类型
 * @param DatatablePath DataTable的资产路径
 * @param DatatableKey 要查找的行名
 * @return 行数据的副本，失败返回默认构造的结构体
 */
template <typename DatatableType>
static FORCEINLINE DatatableType FromDatatable(const FString& DatatablePath, const FName& DatatableKey)
{
    if (DatatablePath == "" || DatatableKey == NAME_None) 
        return DatatableType();
    
    UDataTable* Data = Cast<UDataTable>(
        StaticLoadObject(UDataTable::StaticClass(), nullptr, *DatatablePath)
    );
    
    if (Data) 
    {
        DatatableType* DTD = Data->FindRow<DatatableType>(
            DatatableKey, 
            "FromDatatable Failed to Find", 
            true
        );
        
        if (DTD)
        {
            return *DTD;
        }
    }
    
    return DatatableType();
}

// 使用示例
// 定义DataTable结构体
USTRUCT(BlueprintType)
struct FWeaponData : public FTableRowBase
{
    GENERATED_USTRUCT_BODY()
    
    UPROPERTY(EditAnywhere, BlueprintReadWrite)
    FString WeaponName;
    
    UPROPERTY(EditAnywhere, BlueprintReadWrite)
    float Damage;
    
    UPROPERTY(EditAnywhere, BlueprintReadWrite)
    float FireRate;
};

// 加载数据
FWeaponData WeaponData = FromDatatable<FWeaponData>(
    TEXT("DataTable'/Game/Data/Weapons'"),
    FName("AssaultRifle")
);
```

**复用指数**: ⭐⭐⭐⭐⭐  
**适用场景**: 所有使用DataTable的UE项目

### 1.3 距离计算工具

**用途**：计算Actor之间的距离

**代码**：
```cpp
/**
 * 计算两个Actor之间的距离
 * @param ActorA 第一个Actor
 * @param ActorB 第二个Actor
 * @return 距离，如果任一Actor无效返回-1.f
 */
static FORCEINLINE float GetDistanceBetweenActors(AActor* ActorA, AActor* ActorB)
{
    if (!ActorA || !ActorB) return -1.f;
    return FVector::Dist(ActorA->GetActorLocation(), ActorB->GetActorLocation());
}

/**
 * 检查Actor是否在范围内
 * @param Actor 目标Actor
 * @param Location 中心位置
 * @param Radius 半径
 * @return 是否在范围内
 */
static FORCEINLINE bool IsActorInRange(AActor* Actor, const FVector& Location, float Radius)
{
    if (!Actor) return false;
    return FVector::DistSquared(Actor->GetActorLocation(), Location) <= (Radius * Radius);
}

// 使用示例
float Distance = GetDistanceBetweenActors(Player, Enemy);
bool bInRange = IsActorInRange(Enemy, Player->GetActorLocation(), 1000.f);
```

**复用指数**: ⭐⭐⭐⭐  
**适用场景**: AI感知、范围检测、距离判断

### 1.4 随机数生成器

**用途**：生成随机数和随机向量

**代码**：
```cpp
/**
 * 生成指定范围内的随机向量
 * @param Origin 中心点
 * @param MinDistance 最小距离
 * @param MaxDistance 最大距离
 * @return 随机位置
 */
static FORCEINLINE FVector GetRandomPointAroundLocation(
    const FVector& Origin, 
    float MinDistance, 
    float MaxDistance
)
{
    const float Angle = FMath::FRandRange(0.f, 2.f * PI);
    const float Distance = FMath::FRandRange(MinDistance, MaxDistance);
    
    return FVector(
        Origin.X + Distance * FMath::Cos(Angle),
        Origin.Y + Distance * FMath::Sin(Angle),
        Origin.Z
    );
}

/**
 * 生成随机方向向量
 * @return 单位长度的随机方向
 */
static FORCEINLINE FVector GetRandomDirection()
{
    return FMath::VRand();
}

/**
 * 生成随机延迟时间
 * @param BaseTime 基础时间
 * @param Deviation 偏差范围
 * @return 随机时间
 */
static FORCEINLINE float GetRandomDelay(float BaseTime, float Deviation)
{
    return FMath::FRandRange(BaseTime - Deviation, BaseTime + Deviation);
}

// 使用示例
FVector SpawnLocation = GetRandomPointAroundLocation(PlayerLocation, 500.f, 3000.f);
FVector RandomDir = GetRandomDirection();
float RandomDelay = GetRandomDelay(3.f, 1.f);  // 2-4秒之间
```

**复用指数**: ⭐⭐⭐⭐  
**适用场景**: 生成器系统、随机行为、延迟执行

### 2.4 网络属性复制模板

**用途**：自动同步属性到客户端

**代码**：
```cpp
// 头文件
UCLASS()
class AMyActor : public AActor
{
    GENERATED_BODY()
    
    // 使用ReplicatedUsing的属性
    UPROPERTY(ReplicatedUsing=OnRep_Health, BlueprintReadOnly, Category = "Stats")
    float Health;
    
    UPROPERTY(ReplicatedUsing=OnRep_IsDead, BlueprintReadOnly, Category = "Stats")
    bool IsDead;
    
    // 简单复制属性
    UPROPERTY(Replicated, BlueprintReadOnly, Category = "Stats")
    float MaxHealth;
    
    // RepNotify函数
    UFUNCTION()
    void OnRep_Health();
    
    UFUNCTION()
    void OnRep_IsDead();
    
    // 复制属性声明
    virtual void GetLifetimeReplicatedProps(TArray<FLifetimeProperty>& OutLifetimeProps) const override;
};

// 实现文件
void AMyActor::GetLifetimeReplicatedProps(TArray<FLifetimeProperty>& OutLifetimeProps) const
{
    Super::GetLifetimeReplicatedProps(OutLifetimeProps);
    
    // 复制属性
    DOREPLIFETIME(AMyActor, Health);
    DOREPLIFETIME(AMyActor, MaxHealth);
    DOREPLIFETIME(AMyActor, IsDead);
    
    // 条件复制示例
    DOREPLIFETIME_CONDITION(AMyActor, SomeProperty, COND_OwnerOnly);  // 只复制给拥有者
}

void AMyActor::OnRep_Health()
{
    // 属性变化时调用（只在客户端）
    OnHealthChanged.Broadcast(Health);
    
    // 更新UI
    if (HealthWidget)
    {
        HealthWidget->UpdateHealth(Health);
    }
}

void AMyActor::OnRep_IsDead()
{
    if (IsDead)
    {
        // 播放死亡动画
        PlayDeathAnimation();
    }
}
```

**关键点**：
- `ReplicatedUsing` - 属性变化时触发回调
- `RepNotify` - 只在客户端调用
- `DOREPLIFETIME` - 注册复制属性
- `DOREPLIFETIME_CONDITION` - 条件复制

**复用指数**: ⭐⭐⭐⭐⭐  
**适用场景**: 所有需要网络同步的属性

### 3.1 AI状态机模板

**用途**：管理AI行为状态

**代码**：
```cpp
// 头文件
UENUM(BlueprintType)
enum class EAIState : uint8
{
    Idle,
    Patrol,
    Alert,
    Chase,
    Attack,
    Stunned
};

UCLASS()
class AMyAIController : public AAIController
{
    GENERATED_BODY()
    
    // 当前状态
    UPROPERTY(VisibleAnywhere, BlueprintReadOnly, Category = "AI|State")
    EAIState CurrentState = EAIState::Idle;
    
    // 前一个状态
    UPROPERTY(VisibleAnywhere, BlueprintReadOnly, Category = "AI|State")
    EAIState PreviousState = EAIState::Idle;
    
    // 状态转换函数
    UFUNCTION(BlueprintCallable, Category = "AI|State")
    void SetAIState(EAIState NewState);
    
    // 获取状态
    UFUNCTION(BlueprintPure, Category = "AI|State")
    EAIState GetAIState() const { return CurrentState; }
    
    // 状态变化回调
    UFUNCTION()
    void OnStateChanged(EAIState OldState, EAIState NewState);
    
protected:
    // 黑板键名
    static const FName StateKeyName;
};

// 实现文件
const FName AMyAIController::StateKeyName = TEXT("AIState");

void AMyAIController::SetAIState(EAIState NewState)
{
    // 检查是否相同状态
    if (CurrentState == NewState) return;
    
    // 某些状态下不允许转换（如眩晕状态）
    if (CurrentState == EAIState::Stunned && NewState != EAIState::Idle)
    {
        return;
    }
    
    // 记录状态变化
    EAIState OldState = CurrentState;
    PreviousState = CurrentState;
    CurrentState = NewState;
    
    // 更新黑板
    if (UBlackboardComponent* BB = GetBlackboardComponent())
    {
        BB->SetValueAsEnum(StateKeyName, static_cast<uint8>(NewState));
    }
    
    // 触发状态变化回调
    OnStateChanged(OldState, NewState);
}

void AMyAIController::OnStateChanged(EAIState OldState, EAIState NewState)
{
    // 根据状态变化执行不同逻辑
    switch (NewState)
    {
    case EAIState::Chase:
        // 开始追逐
        if (APawn* Pawn = GetPawn())
        {
            // 设置速度等
        }
        break;
        
    case EAIState::Attack:
        // 开始攻击
        break;
        
    case EAIState::Stunned:
        // 眩晕状态
        break;
    }
}
```

**复用指数**: ⭐⭐⭐⭐⭐  
**适用场景**: 所有需要状态管理的AI系统

### 3.2 AI感知配置模板

**用途**：配置AI的感知能力

**代码**：
```cpp
// 头文件
UCLASS()
class AMyAIController : public AAIController
{
    GENERATED_BODY()
    
protected:
    // 感知组件
    UPROPERTY(VisibleDefaultsOnly, BlueprintReadOnly, Category = "AI|Perception")
    UAIPerceptionComponent* PerceptionComponent;
    
    // 视觉配置
    UPROPERTY(VisibleDefaultsOnly, BlueprintReadOnly, Category = "AI|Perception")
    UAISenseConfig_Sight* SightConfig;
    
    // 听觉配置（可选）
    UPROPERTY(VisibleDefaultsOnly, BlueprintReadOnly, Category = "AI|Perception")
    UAISenseConfig_Hearing* HearingConfig;
    
    // 感知更新回调
    UFUNCTION()
    void OnTargetPerceptionUpdated(AActor* Actor, FAIStimulus Stimulus);
    
public:
    AMyAIController();
};

// 实现文件
AMyAIController::AMyAIController()
{
    // 创建感知组件
    PerceptionComponent = CreateDefaultSubobject<UAIPerceptionComponent>(TEXT("PerceptionComponent"));
    
    // 配置视觉
    SightConfig = CreateDefaultSubobject<UAISenseConfig_Sight>(TEXT("SightConfig"));
    SightConfig->SightRadius = 2000.f;              // 视野半径
    SightConfig->LoseSightRadius = 2500.f;          // 丢失视野半径
    SightConfig->PeripheralVisionAngleDegrees = 90.f; // 周边视野角度
    SightConfig->SetMaxAge(5.f);                    // 记忆持续时间
    
    // 设置检测阵营
    SightConfig->DetectionByAffiliation.bDetectEnemies = true;
    SightConfig->DetectionByAffiliation.bDetectFriendlies = false;
    SightConfig->DetectionByAffiliation.bDetectNeutrals = true;
    
    // 配置感知组件
    PerceptionComponent->ConfigureSense(*SightConfig);
    PerceptionComponent->SetDominantSense(SightConfig->GetSenseImplementation());
    
    // 绑定回调
    PerceptionComponent->OnTargetPerceptionUpdated.AddDynamic(
        this, 
        &AMyAIController::OnTargetPerceptionUpdated
    );
    
    // 可选：配置听觉
    HearingConfig = CreateDefaultSubobject<UAISenseConfig_Hearing>(TEXT("HearingConfig"));
    HearingConfig->HearingRange = 1500.f;
    HearingConfig->SetMaxAge(3.f);
    PerceptionComponent->ConfigureSense(*HearingConfig);
}

void AMyAIController::OnTargetPerceptionUpdated(AActor* Actor, FAIStimulus Stimulus)
{
    // 检查是否成功感知
    if (Stimulus.WasSuccessfullySensed())
    {
        // 目标被感知到
        if (UBlackboardComponent* BB = GetBlackboardComponent())
        {
            BB->SetValueAsObject(TEXT("Target"), Actor);
            BB->SetValueAsVector(TEXT("LastKnownLocation"), Stimulus.StimulusLocation);
        }
        
        // 更新状态
        SetAIState(EAIState::Chase);
    }
    else
    {
        // 目标丢失
        if (UBlackboardComponent* BB = GetBlackboardComponent())
        {
            BB->SetValueAsObject(TEXT("Target"), nullptr);
        }
        
        // 更新状态
        SetAIState(EAIState::Alert);
    }
}
```

**复用指数**: ⭐⭐⭐⭐⭐  
**适用场景**: 所有需要感知系统的AI

void ADynamicSpawner::BeginPlay()
{
    Super::BeginPlay();
    
    // 启动生成计时器
    if (HasAuthority())
    {
        GetWorld()->GetTimerManager().SetTimer(
            SpawnTimer,
            this,
            &ADynamicSpawner::AttemptSpawn,
            SpawnInterval,
            true
        );
    }
}

void ADynamicSpawner::Tick(float DeltaTime)
{
    Super::Tick(DeltaTime);
    
    // 清理已死亡的Pawn引用
    SpawnedPawns.RemoveAll([](const TWeakObjectPtr<APawn>& WeakPtr) {
        return !WeakPtr.IsValid() || WeakPtr->IsPendingKillPending();
    });
    
    CurrentAlive = SpawnedPawns.Num();
}

bool ADynamicSpawner::CanSpawn() const
{
    // 检查并发数量
    if (CurrentAlive >= MaxConcurrent) return false;
    
    // 检查玩家视野
    if (bCheckPlayerVision && IsPlayerLookingAtSpawner()) return false;
    
    return true;
}

bool ADynamicSpawner::IsPlayerLookingAtSpawner() const
{
    // 获取所有玩家
    TArray<ACharacter*> Players;
    for (FConstPlayerControllerIterator It = GetWorld()->GetPlayerControllerIterator(); It; ++It)
    {
        if (APlayerController* PC = It->Get())
        {
            if (ACharacter* Player = Cast<ACharacter>(PC->GetPawn()))
            {
                Players.Add(Player);
            }
        }
    }
    
    // 检查每个玩家是否在看生成器
    for (ACharacter* Player : Players)
    {
        FVector PlayerLocation = Player->GetActorLocation();
        FVector PlayerForward = Player->GetActorForwardVector();
        FVector ToSpawner = (GetActorLocation() - PlayerLocation).GetSafeNormal();
        
        float DotProduct = FVector::DotProduct(PlayerForward, ToSpawner);
        float Angle = FMath::RadiansToDegrees(FMath::Acos(DotProduct));
        
        if (Angle <= PlayerVisionAngle)
        {
            return true;  // 玩家在看生成器
        }
    }
    
    return false;
}

FVector ADynamicSpawner::GetSpawnLocation() const
{
    // 获取最近的玩家
    ACharacter* NearestPlayer = nullptr;
    float NearestDistance = FLT_MAX;
    
    for (FConstPlayerControllerIterator It = GetWorld()->GetPlayerControllerIterator(); It; ++It)
    {
        if (APlayerController* PC = It->Get())
        {
            if (ACharacter* Player = Cast<ACharacter>(PC->GetPawn()))
            {
                float Distance = FVector::Dist(Player->GetActorLocation(), GetActorLocation());
                if (Distance < NearestDistance)
                {
                    NearestDistance = Distance;
                    NearestPlayer = Player;
                }
            }
        }
    }
    
    if (!NearestPlayer) return GetActorLocation();
    
    // 在玩家周围生成
    FVector PlayerLocation = NearestPlayer->GetActorLocation();
    float Angle = FMath::FRandRange(0.f, 2.f * PI);
    float Distance = FMath::FRandRange(MinSpawnDistance, MaxSpawnDistance);
    
    return FVector(
        PlayerLocation.X + Distance * FMath::Cos(Angle),
        PlayerLocation.Y + Distance * FMath::Sin(Angle),
        PlayerLocation.Z
    );
}

void ADynamicSpawner::AttemptSpawn()
{
    if (!CanSpawn()) return;
    
    FVector SpawnLocation = GetSpawnLocation();
    FRotator SpawnRotation = FRotator::ZeroRotator;
    
    FActorSpawnParameters SpawnParams;
    SpawnParams.SpawnCollisionHandlingOverride = ESpawnActorCollisionHandlingMethod::AdjustIfPossibleButAlwaysSpawn;
    
    APawn* SpawnedPawn = GetWorld()->SpawnActor<APawn>(
        PawnClass,
        SpawnLocation,
        SpawnRotation,
        SpawnParams
    );
    
    if (SpawnedPawn)
    {
        SpawnedPawns.Add(SpawnedPawn);
        CurrentAlive++;
        
        // 绑定死亡事件（需要Pawn有死亡事件）
        // SpawnedPawn->OnDeath.AddDynamic(this, &ADynamicSpawner::OnPawnDied);
    }
}

void ADynamicSpawner::ForceSpawn()
{
    AttemptSpawn();
}

void ADynamicSpawner::OnPawnDied(APawn* DeadPawn)
{
    CurrentAlive--;
    SpawnedPawns.Remove(DeadPawn);
}

bool UInventoryComponent::AddItem(FName ItemID, int32 Count)
{
    if (!HasAuthority())
    {
        ServerAddItem(ItemID, Count);
        return true;
    }
    
    FItemData ItemData = GetItemData(ItemID);
    if (ItemData.ItemID == NAME_None) return false;
    
    // 检查是否可堆叠
    if (CanStackItem(ItemID))
    {
        // 查找现有槽位
        int32 ExistingSlot = FindItemSlot(ItemID);
        if (ExistingSlot != INDEX_NONE)
        {
            int32 MaxStack = GetMaxStackSize(ItemID);
            int32 SpaceInStack = MaxStack - Inventory[ExistingSlot].Count;
            int32 ToAdd = FMath::Min(Count, SpaceInStack);
            
            Inventory[ExistingSlot].Count += ToAdd;
            Count -= ToAdd;
            
            OnInventoryUpdated.Broadcast(Inventory, ExistingSlot);
            
            if (Count <= 0) return true;
        }
    }
    
    // 添加到新槽位
    while (Count > 0 && Inventory.Num() < MaxSlots)
    {
        int32 MaxStack = CanStackItem(ItemID) ? GetMaxStackSize(ItemID) : 1;
        int32 ToAdd = FMath::Min(Count, MaxStack);
        
        Inventory.Add(FInventorySlot(ItemID, ToAdd));
        Count -= ToAdd;
        
        OnInventoryUpdated.Broadcast(Inventory, Inventory.Num() - 1);
    }
    
    return Count <= 0;
}

bool UInventoryComponent::RemoveItem(FName ItemID, int32 Count)
{
    if (!HasAuthority())
    {
        ServerRemoveItem(ItemID, Count);
        return true;
    }
    
    if (!HasItem(ItemID, Count)) return false;
    
    for (int32 i = Inventory.Num() - 1; i >= 0 && Count > 0; i--)
    {
        if (Inventory[i].ItemID == ItemID)
        {
            int32 ToRemove = FMath::Min(Count, Inventory[i].Count);
            Inventory[i].Count -= ToRemove;
            Count -= ToRemove;
            
            if (Inventory[i].Count <= 0)
            {
                Inventory.RemoveAt(i);
            }
            
            OnInventoryUpdated.Broadcast(Inventory, i);
        }
    }
    
    return true;
}

bool UQuestManager::StartQuest(FName QuestID)
{
    if (!HasAuthority()) return false;
    
    // 检查任务是否存在
    if (!AllQuests.Contains(QuestID)) return false;
    
    // 检查是否已激活或完成
    if (IsQuestActive(QuestID) || IsQuestCompleted(QuestID)) return false;
    
    // 获取任务数据
    FQuestData Quest = AllQuests[QuestID];
    Quest.State = EQuestState::Active;
    
    // 激活第一个目标
    if (Quest.Objectives.Num() > 0)
    {
        Quest.Objectives[0].State = EObjectiveState::Active;
    }
    
    // 添加到激活列表
    ActiveQuests.Add(Quest);
    
    // 广播事件
    OnQuestStateChanged.Broadcast(QuestID, EQuestState::Active);
    
    return true;
}

void UQuestManager::ReportKill(FName EnemyTag)
{
    if (!HasAuthority()) return;
    
    for (FQuestData& Quest : ActiveQuests)
    {
        if (Quest.State != EQuestState::Active) continue;
        
        for (FQuestObjective& Objective : Quest.Objectives)
        {
            if (Objective.State != EObjectiveState::Active) continue;
            if (Objective.Type != EObjectiveType::Kill) continue;
            
            // 检查目标标签
            if (Objective.TargetTag == NAME_None || Objective.TargetTag == EnemyTag)
            {
                Objective.CurrentCount++;
                
                // 广播进度更新
                OnObjectiveProgressUpdated.Broadcast(
                    Quest.QuestID,
                    Objective.ObjectiveID,
                    Objective.CurrentCount,
                    Objective.TargetCount
                );
                
                // 检查完成
                if (Objective.CurrentCount >= Objective.TargetCount)
                {
                    Objective.State = EObjectiveState::Completed;
                    
                    // 激活下一个目标
                    int32 ObjIndex = Quest.Objectives.Find(Objective);
                    if (Quest.Objectives.IsValidIndex(ObjIndex + 1))
                    {
                        Quest.Objectives[ObjIndex + 1].State = EObjectiveState::Active;
                    }
                    
                    // 检查任务完成
                    CheckQuestCompletion(Quest.QuestID);
                }
            }
        }
    }
}

    OutResult.Rotation = FRotator(Pitch, Yaw, Roll);
}
