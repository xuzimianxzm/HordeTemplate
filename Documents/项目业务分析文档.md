## 1. 项目概述

### 1.1 项目简介

Horde Template V2 (Native) 是一个基于 Unreal Engine 5.7 C++ 开发的**合作僵尸生存游戏框架**。该项目提供了一个完整的、可扩展的游戏框架，包含：

- **完整的AI系统**：智能僵尸AI，具有状态机、感知系统和行为树
- **武器系统**：服务器权威的武器系统，支持多种射击模式
- **物品系统**：可扩展的物品和背包系统
- **任务系统**：灵活的任务管理系统
- **多人游戏支持**：完整的多人联机功能
- **UI系统**：基于UMG的完整UI框架

### 1.2 项目特色

1. **服务器权威架构**：所有关键游戏逻辑都在服务器端验证
2. **高度模块化**：各系统之间低耦合，易于扩展和复用
3. **数据驱动**：大量使用DataTable进行配置
4. **网络优化**：精心设计的网络复制策略
5. **详细的代码注释**：每个类和函数都有完整的文档注释

```cpp
void AZedAIController::SetAIState(EZedAIState NewState)
{
    if (CurrentAIState == NewState) return;
    
    // 不允许在眩晕状态下改变状态（除了眩晕结束）
    if (bIsStaggered && NewState != EZedAIState::Patrol && NewState != EZedAIState::Idle)
        return;
    
    EZedAIState OldState = CurrentAIState;
    PreviousAIState = CurrentAIState;
    CurrentAIState = NewState;
    
    // 更新黑板
    UBlackboardComponent* BB = GetBlackboardComponent();
    if (BB)
    {
        BB->SetValueAsEnum("AIState", static_cast<uint8>(NewState));
    }
    
    OnStateChanged(OldState, NewState);
}
```

```cpp
float AZedAIController::GetTargetSpeedForState(EZedAIState State) const
{
    switch (State)
    {
    case EZedAIState::Idle:

    case EZedAIState::Patrol:
        return ZED_SPEED_PATROL;    // 200 u/s
```

### 3.2 武器系统

#### 3.2.1 服务器权威射击

武器系统采用服务器权威设计，防止作弊：

**射击验证**：
```cpp
bool ABaseFirearm::CanFire() const
{
    // 必须有弹药
    if (LoadedAmmo <= 0) return false;
    
    // 必须通过所有者验证
    if (!ValidateOwnerState()) return false;
    
    // 检查射速时序（服务器权威）
    if (HasAuthority())
    {
        const double CurrentTime = GetWorld()->GetTimeSeconds();
        const double TimeSinceLastShot = CurrentTime - LastFireTime;
        
        if (TimeSinceLastShot < CachedFireRate) return false;
    }
    
    return true;
}
```

**射击执行**：
```cpp
void ABaseFirearm::FireFirearm()
{
    if (!HasAuthority()) return;
    if (!CanFire()) return;
    
    // 更新最后射击时间（防止双发漏洞）
    LastFireTime = GetWorld()->GetTimeSeconds();
    
    // 消耗弹药
    LoadedAmmo = FMath::Max(LoadedAmmo - 1, 0);
    
    // 播放特效
    PlayFirearmFX();
    
    // 应用散布
    FRotator SpreadRotation = ApplySpread(EyeRotation);
    
    // 生成投射物
    ABaseProjectile* Projectile = GetWorld()->SpawnActor<ABaseProjectile>(
        CurrentWeaponItem.ProjectileClass,
        SpawnTransform,
        SpawnParams
    );
    
    // 更新累积散布
    AccumulatedSpread = FMath::Min(AccumulatedSpread + SpreadIncreasePerShot, MaxAccumulatedSpread);
}
```

#### 3.2.2 散布系统

实现真实的武器散布机制：

**散布计算**：
```cpp
float ABaseFirearm::CalculateCurrentSpread() const
{
    float CurrentSpread = BaseSpreadAngle + AccumulatedSpread;
    
    AHordeBaseCharacter* OwnerChar = GetOwningCharacter();
    if (OwnerChar)
    {
        // 移动时增加散布
        if (OwnerChar->GetVelocity().SizeSquared() > 100.0f)
        {
            CurrentSpread *= MovingSpreadMultiplier;
        }
        
        // 空中时增加散布
        UCharacterMovementComponent* MovementComp = OwnerChar->GetCharacterMovement();
        if (MovementComp && !MovementComp->IsMovingOnGround())
        {
            CurrentSpread *= AirborneSpreadMultiplier;
        }
    }
    
    return FMath::Clamp(CurrentSpread, 0.0f, MaxAccumulatedSpread + BaseSpreadAngle * AirborneSpreadMultiplier * MovingSpreadMultiplier);
}
```

**散布应用**：
```cpp
FRotator ABaseFirearm::ApplySpread(const FRotator& BaseRotation) const
{
    const float SpreadAngle = CalculateCurrentSpread();
    
    if (SpreadAngle <= 0.0f) return BaseRotation;
    
    // 在锥形范围内生成随机散布
    const float HalfAngleRad = FMath::DegreesToRadians(SpreadAngle * 0.5f);
    const float RandomAngle = FMath::FRandRange(0.0f, 2.0f * PI);
    const float RandomRadius = FMath::FRandRange(0.0f, FMath::Sin(HalfAngleRad));
    
    const float SpreadYaw = FMath::RadiansToDegrees(FMath::Cos(RandomAngle) * RandomRadius);
    const float SpreadPitch = FMath::RadiansToDegrees(FMath::Sin(RandomAngle) * RandomRadius);
    
    return FRotator(BaseRotation.Pitch + SpreadPitch, BaseRotation.Yaw + SpreadYaw, BaseRotation.Roll);
}
```

```cpp
void ABaseFirearm::ExecuteFireSequence()
{
    if (!HasAuthority()) return;
    
    // 检查是否应该停止射击
    AHordeBaseCharacter* OwnerChar = GetOwningCharacter();
    if (!OwnerChar || OwnerChar->GetIsDead() || OwnerChar->Reloading)
    {
        ServerStopFiring();
        return;
    }
    
    // 检查弹药
    if (LoadedAmmo <= 0)
    {
        ServerStopFiring();
        return;
    }
    
    // 检查射速时序
    const double CurrentTime = GetWorld()->GetTimeSeconds();
    const double TimeSinceLastShot = CurrentTime - LastFireTime;
    if (TimeSinceLastShot < CachedFireRate) return;
    
    // 射击
    FireFirearm();
    
    // 处理点射完成
    if (FiringState == EWeaponFiringState::BurstFiring)
    {
        CurrentBurstCount++;
        if (CurrentBurstCount >= MaxBurstCount)
        {
            OnBurstComplete();
        }
    }
}
```

```cpp
{
    if (!HasAuthority()) return;
    
    for (FQuestData& Quest : ActiveQuests)
    {
        if (Quest.State != EQuestState::Active) continue;
        
        for (FQuestObjective& Objective : Quest.Objectives)
        {
            if (Objective.State != EObjectiveState::Active) continue;
            
            // 检查目标类型
            if (Objective.ObjectiveType == EQuestObjectiveType::KillEnemies)
            {
                Objective.CurrentCount++;
                
                // 广播进度更新
                OnObjectiveProgressUpdated.Broadcast(
                    Quest.QuestID, 
                    Objective.ObjectiveID, 
                    Objective.CurrentCount, 
                    Objective.TargetCount
                );
                
                // 检查完成
                if (Objective.CurrentCount >= Objective.TargetCount)
                {
                    Objective.State = EObjectiveState::Completed;
                    OnObjectiveStateChanged.Broadcast(
                        Quest.QuestID, 
                        Objective.ObjectiveID, 
                        EObjectiveState::Completed
                    );
                    
                    AwardObjectiveRewards(Objective);
                }
            }
            else if (Objective.ObjectiveType == EQuestObjectiveType::KillSpecificType)
            {
                // 检查敌人类型
                if (Enemy->GetClass()->IsChildOf(Objective.TargetEnemyClass))
                {
                    Objective.CurrentCount++;
                    // ... 同上
                }
            }
        }
        
        // 检查任务完成
        CheckQuestCompletion(Quest.QuestID);
    }
```

```cpp
/**
 * 从资产路径动态加载对象
 * @tparam ObjType 要加载的对象类型（如 UTexture2D, USoundCue）
 * @param Path 资产路径作为 FName（如 "Texture2D'/Game/...'"）
 * @return 加载的对象指针，如果未找到则返回 nullptr
 */
template <typename ObjType>
static FORCEINLINE ObjType* ObjectFromPath(const FName& Path)
{
    if (Path == NAME_None) return nullptr;
    return Cast<ObjType>(StaticLoadObject(ObjType::StaticClass(), nullptr, *Path.ToString()));
}

/**
 * 从DataTable按路径和行名检索行
 * @tparam DatatableType DataTable行的结构体类型
 * @param DatatablePath DataTable的资产路径
 * @param DatatableKey 要查找的行名
 * @return 行数据的副本，如果未找到则返回默认构造的结构体
 */
template <typename DatatableType>
static FORCEINLINE DatatableType FromDatatable(const FString& DatatablePath, const FName& DatatableKey)
{
    if (DatatablePath == "" || DatatableKey == NAME_None) return DatatableType();
    
    UDataTable* Data = Cast<UDataTable>(StaticLoadObject(UDataTable::StaticClass(), nullptr, *DatatablePath));
    if (Data) {
        DatatableType * DTD = Data->FindRow<DatatableType>(DatatableKey, "FromDatatable Failed to Find", true);
        if (DTD)
        {
            return *DTD;
        }
        else {
            return DatatableType();
        }
    }
    else {
        return DatatableType();
    }
}
```

**使用示例**：
```cpp
// 加载行为树

// 加载物品数据

```

**复用价值**：⭐⭐⭐⭐⭐
- 高度通用，适用于任何UE项目

### 5.9 平滑插值模式

**位置**：[ZedPawn.h:53-64](file:///d:/Unreal/Projects/HordeTemplate/Source/HordeTemplateV2Native/Public/AI/ZedPawn.h#L53-L64)

**代码**：
```cpp
class AZedPawn : public ACharacter
{
    UPROPERTY(Replicated, BlueprintReadOnly)
    float TargetWalkSpeed = 200.f;
    
    UPROPERTY(BlueprintReadOnly)
    float CurrentWalkSpeed = 200.f;
    
    UFUNCTION(BlueprintCallable)
    void SetTargetWalkSpeed(float NewTargetSpeed);
    
    UFUNCTION()
    void UpdateWalkSpeedInterpolation(float DeltaTime);
};

void AZedPawn::UpdateWalkSpeedInterpolation(float DeltaTime)
{
    // 平滑插值到目标速度
    if (!FMath::IsNearlyEqual(CurrentWalkSpeed, TargetWalkSpeed, 0.1f))
    {
        float InterpSpeed = FMath::IsNearlyZero(TargetWalkSpeed) ? 
            ZED_SPEED_DECELERATION : ZED_SPEED_ACCELERATION;
        
        CurrentWalkSpeed = FMath::FInterpConstantTo(
            CurrentWalkSpeed, 
            TargetWalkSpeed, 
            DeltaTime, 
            InterpSpeed
        );
        
        // 应用到移动组件
        GetCharacterMovement()->MaxWalkSpeed = CurrentWalkSpeed;
    }
}
```

**关键原则**：
1. **目标值与当前值分离**：避免跳跃
2. **使用FInterpConstantTo**：恒定速度插值
3. **每帧更新**：在Tick中平滑过渡

**复用价值**：⭐⭐⭐⭐
- 适用于所有需要平滑过渡的系统
- 提高视觉质量
- 避免突变
